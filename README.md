## 2.BlockQuote ##

이메일에서 사용하는 > 블럭인용문자를 이용한다.

<pre><code> > This is a blockqute. </pre></code>

>This is a blockqute.

이 안에서는 다른 마크다운 요소를 포함할 수 있다.

>This is a blockqute.
